Frequently Asked Questions

What is ERMs prefix?

ERM supports slash commands. We also have the default prefix of ">."

I'm getting an "'NoneType' object has no attribute 'send' " error!

Do you have a logging channel set up for the command module you're trying to run?

How do I get a punishment ID?

Search the user you punished, click the right arrow, and copy the ID of the punishment.

I can't accept/deny an LOA!

ERM denies you access to accept your own LOA/RA.

I'm getting a "invalid literal for int()" error!

To format times, you must put a number before s - (seconds), m - (minutes), h - (hours), or d - (days). It should be formatted like 5d for 5 days.

I'm getting a "Missing Permissions" error!

ERMs role must be higher than the on-duty role, LOA role, and RA role to apply the role.

How do I create a reminder?

To create a reminder, run /reminders add, then follow the prompts.

Do you guys have documentation?

Yep! Check it out - https://docs.ermbot.xyz

Why should I pick ERM over the rest?

ERM by far provides the most features in our market. We do everything the competition does + much more.
